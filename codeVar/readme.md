# 快速搜索翻译后不同格式变量名的 utools 插件
### 功能:
- 翻译你输入的内容为英文

- 提供编程中变量命名的小驼峰, 大驼峰, 下划线, 常量格式的翻译内容

- 只需要上下翻动, 回车即可复制到剪切板


### 下载方式    
- 本插件下载地址:[https://github.com/qiaoanqiao/codevar/releases](https://github.com/qiaoanqiao/codevar/releases)
选择最高版本下载即可

### 升级方式:
如果在之后的使用中. 感觉不爽了.点击插件作者链接, 然后进入插件下载地址. 看看有没有新版本吧
如果你有什么建议欢迎提出
